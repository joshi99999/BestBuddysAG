\doxysubsection{Detailed Description}
An \mbox{\hyperlink{classdetection__node_1_1tracker_1_1EuclideanDistTracker}{Euclidean\+Dist\+Tracker}} instance is used to track and detected Objects, in an image. 

To work properly the following conditions must be met.
\begin{DoxyItemize}
\item The given image must be a binary image.
\item Objects need to be big enough.
\item Objects need to be bright enough.
\item Objects should not contact each other to detect an track them properly 
\end{DoxyItemize}
