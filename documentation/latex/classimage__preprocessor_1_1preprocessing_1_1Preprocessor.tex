\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classimage__preprocessor_1_1preprocessing_1_1Preprocessor}{Preprocessor}} instance is used to create a ROS2 node for image preprocessing. 

The node subscribes the topic \char`\"{}camera\+\_\+stream\char`\"{}, preprocesses incoming images from that topic and publishes the preprocessed images on the topic \char`\"{}preprocessed\+\_\+stream\char`\"{}.

The preprocessing includes\+:
\begin{DoxyItemize}
\item Cutting out the belt area.
\item Transformation of the perspective.
\item Conversion to a binary image. 
\end{DoxyItemize}
