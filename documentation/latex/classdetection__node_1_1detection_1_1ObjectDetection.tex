\doxysubsection{Detailed Description}
An \mbox{\hyperlink{classdetection__node_1_1detection_1_1ObjectDetection}{Object\+Detection}} instance is used to create a ROS2 node for image detection. 

The node subscribes the topic \char`\"{}preprocessed\+\_\+stream\char`\"{}, processes incoming images from that topic and publishes the processed images on the topic \char`\"{}id\+\_\+sample\char`\"{} for classification and publishes the processed images on the topic \char`\"{}id\+\_\+pos\+\_\+time\char`\"{} for modelling.

The detection includes\+:
\begin{DoxyItemize}
\item Detecting objects in the image.
\item Tracking the objects with their positions.
\item Creating samples for classification. 
\end{DoxyItemize}
