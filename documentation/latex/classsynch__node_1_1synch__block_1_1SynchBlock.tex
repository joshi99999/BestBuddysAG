\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classsynch__node_1_1synch__block_1_1SynchBlock}{Synch\+Block}} instance is used to create a ROS2 node for synchronisation. 

The node subscribes the topics \char`\"{}id\+\_\+class\+\_\+vec\char`\"{} from the classification and \char`\"{}id\+\_\+pos\+\_\+vel\+\_\+time\char`\"{} from the modelling. It processes incoming data from those topics and publishes the synchronised message on the topic \char`\"{}pos\+\_\+vel\+\_\+class\char`\"{} for the logic.

The synchronisation includes\+:
\begin{DoxyItemize}
\item Data from the classification.
\item Data from the modelling. 
\end{DoxyItemize}
